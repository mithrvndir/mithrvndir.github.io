# Resume Site — Design Brainstorm

## Approach 1: Swiss Typographic Grid
<response>
<text>
**Design Movement:** International Typographic Style (Swiss Design)
**Core Principles:**
- Strict grid discipline with generous white space
- Typography as the primary visual element
- Monochrome palette with a single warm accent
- Content hierarchy through size and weight alone

**Color Philosophy:** Near-white background (#FAFAF8), near-black text (#1A1A18), warm sand accent (#C8A96E). The restraint communicates confidence.

**Layout Paradigm:** Left-anchored sidebar (fixed, narrow) for navigation; right-side scrollable content. Asymmetric split: 20% / 80%.

**Signature Elements:**
- Thin horizontal rules as section dividers
- Large, oversized section numerals in muted tone

**Interaction Philosophy:** Hover states reveal underlines; no decorative animations.

**Animation:** Subtle fade-in on scroll for section entries.

**Typography System:** `DM Serif Display` for name/headings + `DM Sans` for body. Tight tracking on headings.
</text>

## Approach 2: Editorial Newspaper
<response>
<text>
**Design Movement:** Editorial / Print-Inspired
**Core Principles:**
- Column-based layout mimicking broadsheet newspapers
- Contrast between serif display and sans-serif body
- Ink-on-paper feel with off-white background
- Dense but breathable information density

**Color Philosophy:** Cream (#F5F0E8), charcoal (#2C2C2C), muted red accent (#C0392B). Evokes authority and craft.

**Layout Paradigm:** Two-column editorial grid for experience; full-width masthead header.

**Signature Elements:**
- Masthead-style name treatment with a thin rule below
- Pull-quote style skill highlights

**Interaction Philosophy:** Minimal; content-first. Links use underline-on-hover only.

**Animation:** No entrance animations; instant render for print-like feel.

**Typography System:** `Playfair Display` for name/headings + `Source Serif 4` for body.
</text>
<probability>0.06</probability>
</response>

## Approach 3: Bauhaus Functional Minimalism ← SELECTED
<response>
<text>
**Design Movement:** Bauhaus / Functional Minimalism
**Core Principles:**
- Form follows function — every element earns its place
- Radical whitespace as a structural element
- Sparse use of a single geometric accent color
- Content breathes; nothing competes

**Color Philosophy:** Pure white (#FFFFFF) background, near-black (#111111) text, a single slate-blue accent (#3D5A80). Cool, calm, professional.

**Layout Paradigm:** Single-column centered layout with a maximum content width of 680px. Sections separated by generous vertical rhythm rather than borders. The narrow column forces editorial discipline.

**Signature Elements:**
- Thin left-border accent on section headings (2px slate-blue)
- Monospaced date/label text for metadata

**Interaction Philosophy:** Hover on links reveals a subtle color shift; no decorative effects.

**Animation:** Staggered fade-up on initial load for hero text; sections animate in on scroll.

**Typography System:** `Cormorant Garamond` for the name (display) + `Outfit` for all body/UI text. The contrast between a classical serif name and a geometric sans-serif body creates quiet sophistication.
</text>
<probability>0.09</probability>
</response>

**Selected:** Approach 3 — Bauhaus Functional Minimalism
